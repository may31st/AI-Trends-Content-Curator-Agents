# 🤖 AI Trends Content Curator Agents

## 📝 Project Description

**AI Trends Content Curator Agents** is an automated content creation system using AI to track, analyze, and generate content about the latest AI technology trends.

## Key Features

- **AI Trend Tracking:** Automatically collect and analyze trend data from Reddit, Twitter, HackerNews, and YouTube.
- **Trend Scoring:** Evaluate the hot trend level of AI topics in real-time.
- **Smart Content Strategy:** Decide to create, skip, or promote content based on trend scores.
- **Multi-platform:** Generate content for Blog, Twitter, LinkedIn, and Newsletter with optimized formats for each platform.
- **Performance Tracking:** Analytics on engagement, topic coverage, and performance by platform.
- **Specialized Curator Team:** Multiple AI agents with different expertise (AI Research, Tools, Business, Ethics).

## 🛠️ Workflow

1. **Trend Research** → Search for hot news and AI topics.
2. **Trend Scoring** → Assess engagement potential from multiple sources.
3. **Strategic Evaluation** → Decide CREATE/SKIP/PROMOTE based on strategy.
4. **Content Creation & Posting** → Publish to appropriate platforms.
5. **Update Metrics** → Track engagement and performance.
6. **Analysis & Optimization** → Improve strategy based on results.

## 🏗️ System Architecture

```
📁 Project
├── trends.py              # Trend data collection
├── profiles.py            # Account & metrics management
├── curator.py             # Main AI agents
├── curator_templates.py   # Guidelines & prompts

```
